# AppStack

An iOS application built with Swift and UIKit.

## Overview

AppStack is an iOS application designed for iPhone and iPad devices.

## Requirements

- iOS 17.0 or later
- Xcode 15.0 or later
- Swift 5.0 or later

## Project Structure

```
AppStack/
├── AppStack/
│   ├── AppDelegate.swift       # Application entry point
│   ├── SceneDelegate.swift     # Scene lifecycle management
│   ├── ViewController.swift    # Main view controller
│   ├── Info.plist             # App configuration
│   ├── Base.lproj/
│   │   ├── Main.storyboard    # Main UI storyboard
│   │   └── LaunchScreen.storyboard  # Launch screen
│   └── Assets.xcassets/       # App assets and images
└── AppStack.xcodeproj/        # Xcode project file
```

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/JamieNudge/AppStack.git
   cd AppStack
   ```

2. Open the project in Xcode:
   ```bash
   open AppStack.xcodeproj
   ```

3. Select your target device or simulator

4. Build and run the project (⌘R)

## Features

- Modern iOS app architecture using UIKit
- Scene-based lifecycle management
- Storyboard-based UI design
- Support for iPhone and iPad
